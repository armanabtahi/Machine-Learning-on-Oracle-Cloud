Banff - Intelligent Crypto Currency Platform
Overview
Banff is an intelligent crypto currency platform developed as part of an internship at Vector Institute, University of Toronto. The platform is designed to predict "BUY", "HOLD", and "SELL" actions based on news sentiment and live market data.

Getting Started
To use Banff, you will need to obtain API keys from newsdata.io and Livecoinwatch. You will also need to have access to an Oracle Cloud Infrastructure account and set up the necessary services and resources.

Clone the repository to your local machine.

Install the necessary Python dependencies using pip.

Set your API keys as environment variables in your terminal session or as part of your project configuration file.

Update the configuration files to point to your Oracle Cloud Infrastructure resources and credentials.

Run the application using your preferred Python environment.
